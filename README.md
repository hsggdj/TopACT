# TopACT
Running TopACT on a E13 mouse brain dataset from Open-ST.

TopACT Paper: https://www.nature.com/articles/s41586-024-07563-1
Open-ST Paper: https://www.cell.com/cell/fulltext/S0092-8674(24)00636-6

TopACT need to datasets. A reference data set to train a classifier and a Spatial data to be classified. 

The reference dataset used in this example is a E12.5 mouse embryo from Mouse Cell Atlas project by Zhejiang University, which can be downloaded from https://bis.zju.edu.cn/MCA/data/dge/E12.5DayEmbryo.zip.
